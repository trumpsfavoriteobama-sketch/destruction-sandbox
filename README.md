# Destruction Sandbox

A fun destruction sandbox game built with Godot 4.x. Destroy buildings, drive a tank, fly a plane, and wreak havoc with various weapons.

## Features

- **Destructible Buildings** - Buildings collapse with realistic physics
- **Vehicles**
  - Tank with turret and shell weapons
  - Plane with bombs and missiles
- **Weapons**
  - Revolver
  - Rocket Launcher
  - Tank shells
  - Plane bombs
- **NPCs** - AI characters that flee from the player
- **Open City Map** - Explore and destroy
- **Satisfying Physics** - Roblox+ level graphics with realistic destruction

## Controls

- **WASD** - Move / Vehicle forward/back/strafe
- **Mouse** - Look around / Aim
- **Q** - Scope/Aim (weapons)
- **E** - Switch weapons/enter vehicles
- **Left Click** - Fire
- **Space** - Jump / Vehicle up (plane)
- **Ctrl** - Crouch / Vehicle down (plane)

## Setup

1. Install Godot 4.2+ from https://godotengine.org/download
2. Clone this repository: `git clone https://github.com/trumpsfavoriteobama-sketch/destruction-sandbox.git`
3. Open the project folder in Godot
4. Press F5 to run the game

## System Requirements

- MacBook (Intel or Apple Silicon M1/M2/M3)
- 4GB RAM minimum (8GB recommended)
- 5GB free disk space
- Godot 4.2 or later
