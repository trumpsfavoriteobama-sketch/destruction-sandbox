# Build Instructions

## For MacBook

1. Open Godot 4.2+
2. Import project folder
3. Click "Build" > "Export"
4. Select "macOS" template
5. Choose save location
6. Click "Export"

## Export Settings

- Platform: macOS
- Target: Release (for better performance)
- Architecture: Universal (M1 and Intel compatible)

## Running the Game

- After export, open the `.app` file
- Or run from Godot editor: Press F5

## Sharing

- Zip the `.app` file
- Send to friends
- They can run it directly (no Godot installation needed)
